## Description

BreezeFill makes web forms feel effortless.

It quietly suggests the details you use most often, like your name, email, phone number, portfolio, LinkedIn, GitHub, school, degree, and graduation info, right when a form asks for them. When you enter something new, BreezeFill asks before remembering it, so nothing gets saved behind your back.

Built for modern forms:

- Suggests likely answers for common fields like name, email, phone, links, education, and profile details
- Learns site-specific answers and can reuse them the next time the same field appears
- Works across modern page patterns including iframes, open shadow DOM, and native select fields
- Handles application-style pages with more robust matching for job and internship forms
- Keeps your saved values local to the browser so you stay in control

Great for:

- job and internship applications
- signup and profile forms
- contact and lead forms
- scholarship and school applications
- repetitive admin work across the web

## Review notes

BreezeFill is a Safari web extension packaged inside a standard app. After installing the app, enable the extension in Safari Extensions settings.

Core flow for review:

1. Open the BreezeFill app or extension settings and add profile details.
2. Visit a normal web form in Safari.
3. Focus a field like email, phone, LinkedIn, school, or degree.
4. BreezeFill suggests a saved value.
5. Entering a new value triggers an explicit save prompt so the user chooses whether to remember it.

Data behavior:

- BreezeFill stores saved values locally in browser extension storage.
- BreezeFill does not use a BreezeFill backend.
- BreezeFill does not sell or share saved autofill data with third parties.
