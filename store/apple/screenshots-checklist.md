# Apple Screenshot Checklist

Apple App Store screenshots need to come from the actual signed app running on macOS, iPhone, or iPad. The existing web and Chrome store screenshots in this repo are useful references, but they are not the final App Store screenshots.

## Public links

- Website: `https://breezefill.vercel.app/`
- Privacy: `https://breezefill.vercel.app/privacy/`

## Recommended screenshot set

### iPhone

1. Settings screen with personal profile values filled in
2. Autofill suggestion appearing on an email or phone field in Safari
3. Education fields being suggested on an application-style form
4. Save prompt asking whether to remember a new value

### iPad

1. Full settings screen with profile and education tabs visible
2. Autofill suggestion on a larger application form in Safari

### macOS

1. Popup or settings view showing the saved profile model
2. In-page suggestion on a real Safari form
3. Save prompt on desktop Safari

## Suggested captions

- `Autofill the details you use most`
- `Stay fast on applications and profile forms`
- `Save new answers only when you say yes`
- `Keep your form memory tidy and local`

## Good demo pages to capture

- The local demo form in this repo
- A safe public application-style form you control
- A neutral signup or profile form in Safari
