# Chrome Upload Checklist

## Before upload

- Make sure the extension version in `manifest.json` is final for this release.
- Make sure the public privacy policy URL is hosted and ready.
- Make sure your Chrome Web Store developer account has 2-Step Verification enabled.

## Upload package

- Use the ZIP at `dist/chrome/`
- Confirm `manifest.json` is at the root of the ZIP, not inside another folder

## Listing assets

- Screenshots: `store/chrome/screenshots/`
- Small promo tile: `store/chrome/promo/breezefill-small-promo-440x280.png`
- Listing copy: `store/chrome/listing.md`
- Privacy policy text: `store/chrome/privacy-policy.md`
- Reviewer notes: `store/chrome/review-notes.md`

## Dashboard fields to prepare

- Single purpose description
- Permission justifications
- Privacy disclosures
- Category
- Support URL or official website
- Privacy policy URL
